# 从前端到 AI Agent 开发：完整实战指南

> 一本写给前端工程师的 AI Agent 开发书籍 —— 从大模型基础到生产级 Agent 系统，配套 4 个递进式实战项目与面试题库。

---

## 这本书是写给谁的

你是一名**前端开发者**，会写 TypeScript/JavaScript，熟悉异步、HTTP、流式渲染和组件化 UI，现在想**系统转型 AI Agent 开发**。

你可能有这些困惑：

- LLM、RAG、Agent、MCP、Function Calling……这些概念到底什么关系？
- 网上教程东一块西一块，缺少能串起来的完整知识体系。
- 想做项目，但不知道一个"完善的 Agent 系统"该长什么样、有哪些工程模块。
- 想面试 Agent / AI 应用岗位，不知道会被问什么。

这本书就是为了解决这些问题。读完之后，你应该能够：

1. **建立完整的 Agent 知识体系** —— 从底层原理到上层架构，不留盲区。
2. **独立构建一个完善的 AI Agent 项目** —— 含工具调用、记忆、RAG、多 Agent 协作、评测、可观测、部署全链路。
3. **把前端能力变成优势** —— 流式 UI、全栈集成、产品化体验，是很多后端/算法背景的人不擅长的。
4. **从容应对 Agent 岗位面试** —— 配套分层面试题库 + 参考答案。

## 本书的三个特点

### 1. 双语代码（TypeScript + Python）
每个核心概念都同时给出 **TypeScript** 和 **Python** 实现。TS 让你用熟悉的语言快速上手、做全栈；Python 让你看懂业界主流生态（LangChain / LlamaIndex 等）和算法同事的代码。

### 2. 框架无关 / 多模型
我们**先讲清楚原理，再讲框架**。代码默认通过一层薄薄的统一抽象来调用模型，可在 **Claude、OpenAI、开源模型（Qwen / Llama）** 之间切换，避免被单一厂商或框架锁死。你会理解框架到底替你做了什么，而不是只会调 API。

### 3. 工程导向 + 实战驱动
不止讲"怎么调通"，更讲"怎么上线"：评测、可观测性、成本、安全、部署一个不少。配套 4 个从易到难的完整项目。

> 📌 **关于代码示例的时效性**：大模型 API、框架版本、模型 ID 和价格变化很快。本书代码以讲清原理为目标，具体 API 细节请以官方最新文档为准。书中涉及的模型 ID / 价格等"易变信息"统一记录在 [附录·资源清单](./06-附录/03-资源与工具清单.md) 并在 [约定](#阅读约定) 中说明。

## 阅读约定

为保证全书一致，所有章节遵循以下约定：

| 约定 | 说明 |
|------|------|
| **语言** | 正文简体中文，代码注释中文，标识符英文。 |
| **代码** | 核心示例同时给出 TypeScript 与 Python；语言切换用小标题标明。 |
| **模型默认值** | 示例默认以"某个高能力模型"指代，必要时用占位符 `MODEL_ID`。具体到厂商时：Anthropic 用 `claude-opus-4-8`（自适应思考），OpenAI 用 `gpt-4.1` 系列，开源用 `Qwen` / `Llama`。**所有模型 ID 与价格属易变信息，以 [资源清单](./06-附录/03-资源与工具清单.md) 为准。** |
| **密钥** | 一律从环境变量读取（`ANTHROPIC_API_KEY` / `OPENAI_API_KEY`），**绝不硬编码**。 |
| **章节结构** | 学习目标 → 正文（含图）→ 双语代码 → 前端视角 → 常见坑 → 小结 → 练习 → 延伸阅读。 |
| **图示** | 用 ASCII / Mermaid 表达流程与架构。 |
| **术语** | 首次出现给出中英文，统一收录于 [术语表](./06-附录/01-术语表.md)。 |
